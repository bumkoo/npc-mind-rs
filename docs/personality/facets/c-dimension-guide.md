# C 차원 (Conscientiousness) 완전 가이드 (현행화)

## 개요

C = **Conscientiousness (성실성)**

HEXACO 성격 모델의 6개 차원 중 다섯 번째 차원이다. 사람이 얼마나 조직적이고 근면하며, 자기 행동에 높은 기준을 적용하고 충동을 억제하는지를 측정한다.

| C 수준 | NPC 성향 |
|--------|---------|
| **높음 (+)** | 조직적이고 근면하며, 완벽주의적이고 신중함. 자기 행동에 높은 기준을 적용 |
| **낮음 (-)** | 무질서하고 게으르며, 충동적이고 무모함. 자기 기준이 느슨함 |

---

## 4개 Facet 상세

### Facet 1~3. Organization / Diligence / Perfectionism
- **정의**: 목표 달성을 위한 성실함과 완벽을 추구하는 성향.
- **연결 감정 (Action)**: **Standards (자기 기준)** 증폭.
- **공식**: `standards_amp = avg_c.abs_modifier(0.3)`
- **로직**: `1.0 + |avg_c| * 0.3`. 성실성 평균이 극단적일수록(매우 높거나 낮음) 자신의 행동에 대한 감정적 가중치가 커진다. 이는 엄격한 완벽주의자든, 규범을 무시하는 방랑자든 자신의 행동에 대해 강한 확신(Pride/Shame)을 가짐을 의미한다.

### Facet 4. Prudence (신중함)
- **정의**: 행동 전에 결과를 고려하고 충동을 억제하는 정도.
- **연결 감정 (Event)**: **Distress (고통)** 충동 억제.
- **공식**: `impulse_mod = prudence.neg_modifier(0.3)`
- **로직**: `1.0 - max(0, prudence) * 0.3`. 신중함 점수가 양수(높음)일 때만 즉각적인 고통 감정을 억제하며, 신중하지 못한(음수, 충동적) 경우에는 억제 효과 없이 기준 강도(1.0)를 유지한다.

---

## C 평균 (avg.c) 및 전역 규칙

```
avg.c = (organization + diligence + perfectionism + prudence) / 4
standards_amp = 1.0 + avg.c.abs() * 0.3
```

- **자기 기준 브레이크**: `avg.c`는 `Pride`, `Shame`, `Admiration`, `Reproach` 등 행동 기반 감정 전반을 증폭하거나 억제하는 **도덕적 기준**으로 작용한다. 
- **양방향 증폭**: C 역시 절댓값(`abs()`)을 사용한다. 성실함이 높든 낮든(극도로 자유분방한 경우 포함), 자신의 행동에 대한 감정적 무게가 커진다.

---

## 대화 중 자극 처리 (Stimulus Engine)

C 차원, 특히 신중함(`Prudence`)은 대화 중 감정이 요동치는 것을 막는 **감정 급변 브레이크** 역할을 한다.

- **급변 억제**: `rate -= prudence.max(0.0) * 0.3`
- **설명**: 신중한 NPC는 대화 중 도발이나 칭찬에 휘둘리지 않는다. 감정 수치가 등락하는 속도 자체를 억제하여, 심리적 평정심을 유지하는 경향을 보인다.

---

## 감정의 소멸 (Fade)

- **C의 영향**: 신중함이 높은 NPC는 감정이 생길 때도 천천히 생기고, 소멸할 때도 천천히 소멸한다. 
- **소멸 임계값**: 감정 강도가 **0.05 미만**으로 떨어지면 시스템에서 삭제된다. 신중한 NPC는 `Distress` 등의 초기 생성이 억제되어 있으므로, 자극이 조금만 반대 방향으로 작용해도 금방 이 소멸 임계값 아래로 떨어져 감정이 해소된다.
