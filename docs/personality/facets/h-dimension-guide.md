# H 차원 (Honesty-Humility) 완전 가이드 (현행화)

## 개요

H = **Honesty-Humility (정직-겸손성)**

HEXACO 성격 모델의 6개 차원 중 첫 번째 차원이다. 사람이 도덕적·사회적 관계에서 얼마나 진실하고 겸손하며, 이익 앞에서 공정한지를 측정한다.

| H 수준 | NPC 성향 |
|--------|---------|
| **높음 (+)** | 진실하고 공정하며, 욕심 없고, 자기를 낮추는 성향 |
| **낮음 (-)** | 교활하고 불공정하며, 물질 집착적이고, 자기 과시를 즐기는 성향 |

---

## 4개 Facet 상세

### Facet 1. Sincerity (진실성)
- **정의**: 자신의 감정과 의도를 얼마나 솔직하게 드러내는가.
- **연결 감정 (Action)**: **Gratitude (감사)** 증폭.
- **공식**: `gratitude_amp = 1.0 + sincerity.max(0.0) * 0.3`
- **설명**: 진실한 NPC일수록 타인의 호의를 왜곡 없이 받아들여 더 깊은 감사를 느낀다.

### Facet 2. Fairness (공정성)
- **정의**: 규칙을 지키고 부정한 이득을 거부하는 정도.
- **연결 감정**: 단독 연결 없음. `avg.h`를 통해 타인의 운(Fortune-of-others) 평가에 참여.

### Facet 3. Greed Avoidance (탐욕 회피)
- **정의**: 부와 사회적 지위에 집착하지 않는 정도.
- **연결 감정**: 단독 연결 없음. `avg.h`를 낮추어 **Resentment**(시기) 발생 가능성을 높임.

### Facet 4. Modesty (겸손)
- **정의**: 자신의 능력과 업적을 과장하지 않고 낮추는 정도.
- **연결 감정 (Action)**: **Pride (자부심)** 억제.
- **공식**: `pride_mod = 1.0 - modesty.max(0.0) * 0.3`
- **설명**: 겸손한 NPC는 큰 성취를 이루어도 자부심을 내세우지 않는다.

---

## H 평균 (avg.h) 및 전역 규칙

```
avg.h = (sincerity + fairness + greed_avoidance + modesty) / 4
```

### 1. 타인의 운 (Fortune-of-others) 평가
- **HappyFor (대리기쁨)**: `(avg.h.max(0.0) + avg.a.max(0.0)) / 2.0` 수치가 공감 강도에 기여.
- **Resentment (시기)**: `avg.h < -0.2` (FORTUNE_THRESHOLD) 일 때만 발생. 강도는 `avg.h.abs()`에 비례.
- **Gloating (고소함)**: `avg.h < -0.2` 이면서 `avg.a < -0.2` 일 때 발생.

### 2. 가이드 레이어 (Acting Guide) 연동
- **Tone**: 자부심(`Pride`)을 느낄 때 `avg.h > 0.3`이면 '조용한 자신감', 이하이면 '거만한 자신감' 어조 사용.
- **Restriction**: `avg.h > 0.5`인 NPC는 감정 상태와 무관하게 **거짓말 및 과장 금지** 제약을 받음.

## 핵심 수치 및 상수

| 항목 | 공식 / 값 | 설명 |
|------|-----------|------|
| **W** | 0.3 | 성격 가중치 범용 계수 |
| **rel_mul** | `1.0 + closeness.intensity() * 0.5` | 관계 친밀도에 따른 전반적 증폭 |
| **trust_mod** | `1.0 + trust.value() * 0.3` | 신뢰도에 따른 행동 감정 보정 |
| **EMPATHY_BASE**| 0.5 | 공감 감정의 기본 발생 강도 |
| **FORTUNE_THRESHOLD**| -0.2 | 시기/고소함 발생을 위한 최소 악성 수치 |
