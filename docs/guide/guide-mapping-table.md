# 성격×감정 → 연기 가이드 매핑표 (최종 현행화)

## 개요

동일한 상황에서도 NPC의 HEXACO 성격에 따라 OCC 감정이 달라지고, 그 감정이 LLM 연기 가이드의 어조, 태도, 행동, 금지사항을 결정한다. 이 문서는 가이드 생성의 정성적 기준(라벨링)과 실제 도출 규칙을 정리한다.

---

## 1. 정성적 라벨링 기준 (presentation/locale.rs)

수치 데이터(f32)를 LLM이 이해하기 쉬운 텍스트 라벨로 변환하는 기준이다.

### 감정 강도 (Intensity)
- **>= 0.8**: 극도로 강한
- **>= 0.6**: 강한
- **>= 0.4**: 뚜렷한
- **>= 0.2**: 약한
- **< 0.2**: 미미한 (연기 지시에 미반영)

### 기분 (Mood / Overall Valence)
- **> 0.5**: 매우 긍정적
- **> 0.2**: 긍정적
- **> -0.2**: 중립적
- **> -0.5**: 부정적
- **<= -0.5**: 매우 부정적

---

## 2. 캐릭터 성격 스냅샷 및 발동 조건

성격 차원 평균(avg)이 **±0.3 (TRAIT_THRESHOLD)**을 초과할 때만 해당 특성과 말투가 가이드에 포함된다.

| 캐릭터 | 핵심 성격 특성 (PersonalityTrait) | 말투 스타일 (SpeechStyle) |
|--------|-----------------------------------|---------------------------|
| **무백** | 진실되고 공정하며 겸손함. 대담하고 독립적임. 관용적이고 온화함. 체계적이고 신중함. | 솔직하고 꾸밈없이. 차분하고 담담하게. 부드럽고 배려 깊은 어조. 논리적이고 이성적임. |
| **교룡** | 교활하고 야심적임. 자신감 있고 사교적임. 원한 품기 쉽고 참을성 없음. 호기심 많고 창의적임. | 속내를 감추고 거짓을 섞음. 적극적이고 목소리에 힘. 날카롭고 직설적임. 비유와 독특한 표현. |
| **수련** | 진실되고 공정함. 관용적이고 인내심 강함. 체계적이고 극도로 신중함. 전통을 존중함. | 솔직하고 꾸밈없이. 부드럽고 배려 깊음. 논리적이고 이성적임. 격식을 차린 전통적 표현. |
| **소호** | 대담하고 독립적임. 자신감 있고 사교적임. 자유분방하고 충동적임. 호기심 많고 창의적임. | 차분하고 담담함. 적극적이고 목소리에 힘. 거침없이 말하고 즉흥적임. 비유와 독특한 표현. |

## 3. HEXACO 차원 → 연기 지시 결정 규칙 (directive.rs)

### 어조 (Tone)
- **Anger** 기반: `C > 0.3` -> 억누른 분노 / `C <= 0.3` -> 거친 공격
- **Distress** 기반: `E > 0.3` -> 불안하고 떨림 / `E <= 0.3` -> 침울하지만 절제됨
- **Fear** 기반: `E < -0.3` -> 경계하지만 침착 / `E >= -0.3` -> 긴장되고 불안
- **Pride** 기반: `H > 0.3` -> 조용한 자신감 / `H <= 0.3` -> 자랑스럽고 거만

### 태도 (Attitude)
- **Anger** 활성 시: `A < -0.3` -> 적대적 공격 / `A >= -0.3` -> 불만 억누른 불편함
- **기본 태도**: Mood 수치(>0.3 / <-0.3)에 따라 호의적/방어적 태도 분기

### 행동 경향 (Behavioral Tendency)
- **Anger** 활성 시: `C < -0.3` -> 즉각 맞섬 / `C > 0.3` -> 전략적 대응 / `기타` -> 표현 후 관찰
- **Fear** 활성 시: `E < -0.3` -> 용기 있게 맞섬 / `E >= -0.3` -> 안전 확보 우선

## 4. ActingGuide 실제 데이터 구조 (JSON)

```json
{
  "npc_name": "교룡",
  "npc_description": "야심적인 여검객...",
  "personality": {
    "traits": "교활하고 야심적이며 자기과시적인 성격이다...",
    "speech_style": "속내를 쉽게 드러내지 않고, 필요하면 거짓도 섞는다..."
  },
  "emotion": {
    "composition": [
      "분노(극도로 강한, 지배) — 교룡의 밀고 행위",
      "비난(강한) — 밀고로 인한 추방 위기"
    ],
    "mood": -0.75,
    "mood_label": "매우 부정적"
  },
  "directive": {
    "tone": "거칠고 공격적인 어조",
    "attitude": "적대적이고 공격적인 태도",
    "behavioral_tendency": "화를 표출하되, 상황을 지켜본다",
    "restrictions": ["농담이나 가벼운 말투를 사용하지 않는다"]
  },
  "situation_description": "동료 무사가 적에게 아군의 위치를 밀고했다",
  "relationship": {
    "target_name": "mu_baek",
    "closeness": "가까운",
    "trust": "신뢰하는",
    "power": "대등한 관계"
  }
}
```
