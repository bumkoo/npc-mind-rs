# 성격×감정 → 연기 가이드 매핑표

## 개요

동일한 상황에서도 NPC의 HEXACO 성격에 따라 OCC 감정이 달라지고,
그 감정이 LLM 연기 가이드의 어조, 태도, 행동, 금지사항을 결정한다.

이 문서는 4인 캐릭터 × 4개 시나리오에서 실제로 생성되는 가이드를 정리한다.

---

## 캐릭터 성격 요약

| 캐릭터 | 핵심 HEXACO | 성격 특성 요약 | 말투 요약 |
|--------|-------------|---------------|-----------|
| 무백 | H↑ A↑ C↑ E↓ | 진실되고 공정하며 겸손. 대담하고 독립적. 관용적이고 인내심 강함. 체계적이고 신중 | 솔직하고 꾸밈없이. 차분하고 담담하게. 부드럽고 배려 깊은 어조. 논리적 |
| 교룡 | H↓ A↓ X↑ O↑ | 교활하고 야심적. 자신감 있고 활기참. 원한 품기 쉽고 참을성 없음. 호기심 많고 비관습적 | 속내를 안 드러냄. 적극적이고 목소리에 힘. 날카롭고 거침없이. 비유 즐김 |
| 수련 | H↑ A↑ C↑↑ O↓ | 진실되고 공정. 관용적이고 인내심 강함. 체계적이고 극도로 신중. 전통 존중 | 솔직. 차분. 부드럽고 배려. 논리적. 격식을 차림 |
| 소호 | E↓ X↑ C↓ O↑ | 대담하고 독립적. 자신감 있고 사교적. 자유분방하고 충동적. 호기심 많고 비관습적 | 차분하고 담담. 적극적이고 목소리에 힘. 거침없이 말하고 즉흥적. 비유 즐김 |

## 시나리오 1: 동료의 배신

**상황**: Action(타인, praiseworthiness=-0.7, outcome_for_self=-0.6)

| 항목 | 무백 | 교룡 | 수련 | 소호 |
|------|------|------|------|------|
| **핵심 성격 작용** | patience=+0.8 → 브레이크 | patience=-0.7 → 증폭 | patience=+0.9 → 극도 억제 | prudence=-0.5 → 충동적 |
| **지배 감정** | 비난(뚜렷한) | 분노(극도로 강한) | 비난(뚜렷한) | 분노(강한) |
| **활성 감정** | 비난, 분노 | 분노, 비난 | 비난, 분노(약한) | 분노, 비난 |
| **분위기** | 부정적 | 매우 부정적 | 부정적 | 부정적 |
| **어조** | 억누른 분노의 차가운 어조 | 거칠고 공격적인 어조 | 억누른 분노의 차가운 어조 | 거칠고 공격적인 어조 |
| **태도** | 불만을 억누르지만 불편함 | 적대적이고 공격적 | 불만을 억누르지만 불편함 | 적대적이고 공격적 |
| **행동 경향** | 분노를 억누르고 계획적 대응 | 화를 표출하되 상황 관찰 | 분노를 억누르고 계획적 대응 | 즉각적으로 맞서고 행동 |
| **금지 사항** | 농담 금지, 호의 금지, 거짓 금지 | 농담 금지, 호의 금지 | 농담 금지, 호의 금지, 거짓 금지 | 농담 금지, 호의 금지 |

**핵심 차이점**:
- 무백/수련은 C↑로 인해 "억누른" 분노 → 차가운 어조, 계획적 대응
- 교룡은 A↓로 인해 "폭발적" 분노 → 거친 어조, 적대적 태도
- 소호는 C↓로 인해 "충동적" 분노 → 즉각 행동으로 옮김
- 무백/수련은 H↑로 인해 "거짓 금지" 제약이 추가됨

## 시나리오 2: 라이벌의 무림맹주 추대

**상황**: Event(desirability_self=0.0, desirability_other=Some(0.8))

| 항목 | 무백 | 교룡 |
|------|------|------|
| **핵심 성격 작용** | H=+0.65, A=+0.58 → 공감 | H=-0.55, A=-0.55 → 질투 |
| **지배 감정** | 대리기쁨(뚜렷한) | 시기(뚜렷한) |
| **분위기** | 긍정적 | 부정적 |
| **어조** | 편안하고 온화한 어조 | 질투가 묻어나는 씁쓸한 어조 |
| **태도** | 호의적이고 개방적 | 방어적이고 닫힌 태도 |
| **행동 경향** | 대화에 적극 참여, 협조 | 상황을 관찰하며 필요한 만큼만 반응 |
| **금지 사항** | (없음) | 농담 금지 |

**핵심 차이점**:
- 같은 "타인의 좋은 일"인데 H의 부호가 반대 → 완전히 다른 감정(대리기쁨 vs 시기)
- 이로 인해 어조, 태도, 분위기가 정반대

## 시나리오 3: 적의 대군이 다가옴 (미래 위협)

**상황**: Event(desirability_self=-0.7, is_prospective=true)

| 항목 | 무백 | 교룡 | 수련 | 소호 |
|------|------|------|------|------|
| **핵심 성격 작용** | E=-0.38 → 담담 | E=0.0 → 중립 | E=+0.03 → 약간 민감 | E=-0.40 → 대담 |
| **지배 감정** | 두려움(뚜렷한) | 두려움(강한) | 두려움(강한) | 두려움(뚜렷한) |
| **분위기** | 부정적 | 부정적 | 부정적 | 부정적 |
| **어조** | 경계하지만 침착한 어조 | 긴장되고 불안한 어조 | 긴장되고 불안한 어조 | 경계하지만 침착한 어조 |
| **태도** | 경계하며 방어적 | 경계하며 방어적 | 경계하며 방어적 | 경계하며 방어적 |
| **행동 경향** | 두려움에도 맞서려 함 | 위험을 피하고 안전 확보 | 위험을 피하고 안전 확보 | 두려움에도 맞서려 함 |
| **금지 사항** | 농담 금지, 허세 금지 | 농담 금지, 허세 금지 | 농담 금지, 허세 금지 | 농담 금지, 허세 금지 |

**핵심 차이점**:
- 무백/소호(E↓)는 "경계하지만 침착" → 맞서려 함 (대담한 성격)
- 교룡/수련(E≈0)은 "긴장되고 불안" → 안전 확보 우선
- E 차원이 두려움 상황에서의 어조와 행동 경향을 직접 결정

## 시나리오 4: 좋은 소식 (동료 귀환)

**상황**: Event(desirability_self=+0.6)

| 항목 | 무백 | 교룡 | 소호 |
|------|------|------|------|
| **핵심 성격 작용** | X=0.0 → 중립 | X=+0.53 → 긍정 증폭 | X=+0.55 → 긍정 증폭 |
| **지배 감정** | 기쁨(뚜렷한) | 기쁨(강한) | 기쁨(강한) |
| **분위기** | 긍정적 | 긍정적 | 긍정적 |
| **어조** | 밝고 활기찬 어조 | 밝고 활기찬 어조 | 밝고 활기찬 어조 |
| **태도** | 호의적이고 개방적 | 호의적이고 개방적 | 호의적이고 개방적 |
| **행동 경향** | 대화에 적극 참여, 협조 | 대화에 적극 참여, 협조 | 대화에 적극 참여, 협조 |
| **금지 사항** | (없음) | (없음) | (없음) |

**핵심 차이점**:
- 긍정 상황에서는 4인 모두 비슷한 가이드 → 이건 자연스러운 결과
- X↑(교룡, 소호)이 기쁨 강도를 증폭시키지만, 연기 지시의 정성적 차이는 크지 않음
- 부정 상황에서 성격 차이가 훨씬 극적으로 드러남

## HEXACO 차원 → 연기 지시 결정 규칙

### 어조(tone) 결정

| 지배 감정 | HEXACO 조건 | 어조 |
|-----------|-------------|------|
| Anger | C > 0.3 (성실↑) | 억누른 분노가 느껴지는 차가운 어조 |
| Anger | C ≤ 0.3 | 거칠고 공격적인 어조 |
| Distress | E > 0.3 (정서↑) | 불안하고 떨리는 어조 |
| Distress | E ≤ 0.3 | 침울하지만 절제된 어조 |
| Fear | E < -0.3 (대담) | 경계하지만 침착한 어조 |
| Fear | E ≥ -0.3 | 긴장되고 불안한 어조 |
| Pride | H > 0.3 (겸손↑) | 조용한 자신감이 묻어나는 어조 |
| Pride | H ≤ 0.3 | 자랑스럽고 거만한 어조 |
| Joy | - | 밝고 활기찬 어조 |
| Shame | - | 움츠러들고 작아진 어조 |
| Reproach | - | 냉소적이고 비판적인 어조 |
| Disappointment | - | 깊은 한숨이 섞인 어조 |
| Gratitude | - | 진심 어린 따뜻한 어조 |
| Resentment | - | 질투가 묻어나는 씁쓸한 어조 |
| Pity | - | 안타까움이 담긴 부드러운 어조 |
| (기타) | mood > 0.2 | 편안하고 온화한 어조 |
| (기타) | mood < -0.2 | 무거운 어조 |
| (기타) | 그 외 | 담담한 어조 |

### 태도(attitude) 결정

| 조건 | HEXACO 보조 조건 | 태도 |
|------|------------------|------|
| Anger 활성 | A < -0.3 (원만성↓) | 적대적이고 공격적인 태도 |
| Anger 활성 | A ≥ -0.3 | 불만을 억누르지만 불편함이 드러나는 태도 |
| Reproach 활성 | - | 상대를 판단하고 평가하는 태도 |
| Fear 활성 | - | 경계하며 방어적인 태도 |
| mood > 0.3 | - | 호의적이고 개방적인 태도 |
| mood < -0.3 | - | 방어적이고 닫힌 태도 |
| 그 외 | - | 중립적이고 관망하는 태도 |

### 행동 경향(behavioral_tendency) 결정

| 조건 | HEXACO 보조 조건 | 행동 경향 |
|------|------------------|-----------|
| Anger 활성 | C < -0.3 (성실↓) | 즉각적으로 맞서고 행동으로 옮기려 한다 |
| Anger 활성 | C > 0.3 (성실↑) | 분노를 억누르고 계획적으로 대응하려 한다 |
| Anger 활성 | 그 외 | 화를 표출하되, 상황을 지켜본다 |
| Fear 활성 | E < -0.3 (대담) | 두려움에도 불구하고 맞서려 한다 |
| Fear 활성 | E ≥ -0.3 | 위험을 피하고 안전을 확보하려 한다 |
| Shame 활성 | - | 화제를 돌리거나 자리를 피하려 한다 |
| mood > 0.3 | - | 대화에 적극적으로 참여하고 협조한다 |
| 그 외 | - | 상황을 관찰하며 필요한 만큼만 반응한다 |

### 금지 사항(restrictions) 결정

| 조건 | 금지 사항 |
|------|-----------|
| mood < -0.3 | 농담이나 가벼운 말투를 사용하지 않는다 |
| Anger 활성 | 상대에게 호의적으로 대하지 않는다 |
| Shame 활성 | 자신의 행동을 정당화하지 않는다 |
| Fear 활성 | 허세를 부리거나 강한 척 하지 않는다 |
| H > 0.5 (정직↑) | 거짓말이나 과장을 하지 않는다 |

금지 사항은 **누적**된다. 여러 조건이 동시에 충족되면 여러 금지 사항이 함께 적용된다.

```

[성격]

[현재 감정]

[상황]
동료 무사가 적에게 아군의 위치를 밀고했다

[연기 지시]

행동 경향: 화를 표출하되, 상황을 지켜본다.
```

## 핵심 관찰

1. **부정 상황에서 성격 차이가 극대화된다**: 같은 배신 상황에서 무백(차가운 절제)과 교룡(폭발적 공격)은 완전히 다른 NPC로 연기됨
2. **긍정 상황에서는 차이가 줄어든다**: 좋은 소식에 대한 반응은 4인 모두 비슷 → 갈등과 위기가 캐릭터성을 드러냄
3. **어조는 감정이, 행동은 성격이 결정한다**: 지배 감정이 어조를 직접 결정하고, HEXACO(특히 C, E)가 행동 경향을 결정
4. **금지 사항이 NPC 일관성을 보장한다**: H↑이면 항상 "거짓 금지", Anger 상태면 항상 "호의 금지" → LLM이 탈선하지 않도록 가드레일 역할
